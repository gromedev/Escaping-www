Graph API Permissions Quick Reference

Delegated vs Application Permissions

Delegated permissions are used when an app acts on behalf of a signed-in user. Application permissions are used when an app runs as a background service without a signed-in user.

Common Permission Scopes

ScopeTypeDescription
User.ReadDelegatedSign in and read user profile
User.Read.AllApplicationRead all users' full profiles
Mail.ReadDelegatedRead user mail
Directory.Read.AllApplicationRead directory data
Group.ReadWrite.AllApplicationRead and write all groups

Admin Consent Required Scopes

Scopes ending in .All typically require admin consent. User.Read and Mail.Read can be user-consented. Directory and Group scopes always require admin consent.
